Self-hosted Webfonts

What do designers do if they would like to use a font that is not web safe but available in their library. How does the client browsers render this font?

For instance, Founders Grotesk is a beautiful font that I would like to use on a web page.

Any font in the format of OTF or TTF can be hosted online, then the client browser would use that resource to render the font on the screen, however there are tweo key issues:

- Legal distribution rights of the font
- Browser incompatiblity

Webfonts

Apparently the most commonly used service for hosting the font on the web is provided by Google, it is called Google Fonts but it is not a monopoly, Typekit and Typotheque are two popular akterntives but keep in mind that this is not a new technology it is just about being able to host a font legally and serving it on demand, so most indy small foundries have similar services also
